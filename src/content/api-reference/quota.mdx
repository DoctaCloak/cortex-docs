# Quota & Pricing API

## Get Usage Quota

```http
GET /api/quota
Authorization: Bearer TOKEN
```

### Response `200`

```json
{
  "tier": "student",
  "queries_used": 142,
  "queries_limit": -1,
  "sonnet_queries_used": 8,
  "sonnet_queries_limit": 20,
  "voice_used": 3,
  "voice_limit": 15,
  "resets_at": "2025-04-01T00:00:00Z"
}
```

A limit of `-1` means unlimited.

---
