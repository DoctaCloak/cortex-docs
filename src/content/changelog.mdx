# Changelog

## v0.3.0 — March 2025

### Added
- **Workspace UI** — Zettelkasten note editor with enforced rules (atomic notes, declarative titles)
- **Activity Stream** — Real-time SSE events for note creation, linking, recall, and model usage
- **Smart Model Routing** — Auto-escalation from Haiku to Sonnet for complex queries
- **Pricing Tiers** — Free, Student ($5), Pro ($12), Power ($30) with metered quotas
- **Voice Responses** — ElevenLabs TTS integration (Student tier and above)

### Changed
- Demo page is now a self-contained product showcase (no backend dependency)
- Default AI model changed from Sonnet to Haiku for cost efficiency
- Conditional database restore on container start (preserves existing data)

## v0.2.0 — February 2025

### Added
- **Graph View** — Interactive knowledge graph visualization
- **Health Monitor** — Graph health monitoring and maintenance tools
- **Recall** — AI-powered semantic search
- **Import/Export** — Import/export for knowledge graphs
- **Demo mode** — Unauthenticated demo with sample data

## v0.1.0 — January 2025

### Added
- Initial release
- SQLite + FTS5 note storage
- ChromaDB vector embeddings
- Basic CRUD API
- GitHub OAuth authentication
- Litestream replication to Cloudflare R2
